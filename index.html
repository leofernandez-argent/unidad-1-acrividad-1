1. ¿Qué es un Sistema Operativo?

Es un software fundamental que gestiona los recursos del hardware y proporciona servicios a los programas de aplicación. Actúa como intermediario entre el usuario y el hardware del computador.

2. Componentes de un Sistema Operativo

- Núcleo (Kernel): Controla el hardware y gestiona procesos.
- Gestor de procesos: Coordina la ejecución de programas.
- Gestor de memoria: Administra la memoria principal.
- Sistema de archivos: Controla el almacenamiento y acceso a archivos.
- Interfaz de usuario: Puede ser gráfica (GUI) o de línea de comandos (CLI).
- Controladores (Drivers): Permiten la comunicación con dispositivos de hardware.

3. Tareas del Sistema Operativo

Ejecutar aplicaciones, administrar hardware, controlar la seguridad del sistema y facilitar la comunicación entre software y hardware.

4. Funciones de un Sistema Operativo

Gestión de procesos, memoria, dispositivos, archivos e interfaz de usuario.

5. Características de un Sistema Operativo

Multitarea, multiusuario, seguridad, estabilidad, escalabilidad y compatibilidad con hardware y software.

6. Tipos de Sistema Operativo

Monousuario y multiusuario, monotarea y multitarea, centralizado y distribuido, tiempo real y sistemas embebidos.

7. Sistemas Operativos Para PC

Microsoft Windows, macOS (Apple) y Linux (varias distribuciones como Ubuntu, Debian, Fedora).

8. Sistemas Operativos Móviles

Android, iOS, HarmonyOS y KaiOS.

9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

PC doméstico: Windows, Linux, macOS. Servidores: Linux Server, Windows Server. Dispositivos móviles: Android, iOS. Sistemas empotrados: RTOS, FreeRTOS, VxWorks.

10. Compatibilidad de los Sistemas Operativos

Depende del tipo de arquitectura (x86, ARM), controladores y aplicaciones soportadas. Algunos sistemas pueden ejecutar software de otros mediante emulación o máquinas virtuales.

11. ¿Cuál es la función básica de un Sistema Operativo?

Gestionar los recursos del sistema y facilitar la interacción entre el usuario y el hardware.

12. Mencione tres Sistemas Operativos más utilizados

Windows, macOS y Linux.

13. Menciona algunas tareas que realizan los Sistemas Operativos

Asignar recursos, coordinar tareas, proveer interfaz de usuario, administrar archivos y controlar periféricos.

14. ¿Qué es un sistema operativo multitareas?

Es aquel que puede ejecutar múltiples procesos o tareas al mismo tiempo, como escuchar música mientras navegas por internet.

15. ¿Qué es un sistema operativo multiusuario?

Permite que varios usuarios utilicen el sistema simultáneamente, como en servidores o sistemas compartidos.

16. ¿Cómo funciona un sistema operativo?

Administra el hardware, ejecuta programas, gestiona memoria y proporciona servicios básicos para el funcionamiento del sistema y las aplicaciones.

17. ¿Cómo se utiliza un sistema operativo en un entorno gráfico?

A través de una Interfaz Gráfica de Usuario (GUI) que permite usar ventanas, iconos y menús para interactuar con el sistema.

18. ¿Cuándo inicia el funcionamiento del Sistema Operativo?

Justo después de encender el equipo y completar el proceso de arranque (boot), el sistema operativo se carga en memoria desde el disco.

19. Mencione las dos interfaces que forman parte del Sistema Operativo

Interfaz gráfica (GUI) e interfaz de línea de comandos (CLI).

20. ¿Qué es un proceso en entorno de los sistemas operativos?

Es una instancia de un programa en ejecución que consume recursos como CPU y memoria.

21. ¿Qué es un sistema operativo distribuido y centralizado?

Distribuido: el sistema opera en múltiples computadoras conectadas, actuando como una sola. Centralizado: todas las operaciones se ejecutan en una sola máquina o servidor principal.

22. Mencione los sistemas de archivos utilizados por los Sistemas Operativos

NTFS, FAT32 (Windows), APFS, HFS+ (macOS), ext4, Btrfs (Linux), exFAT (compatibilidad cruzada).

23. Mencione otras de las características actuales de los sistemas operativos

Soporte para virtualización, actualizaciones automáticas, integración en la nube, seguridad avanzada, cifrado, e interfaces táctiles.

24. ¿Qué permite la exigencia de muchos controladores en un sistema operativo?

La compatibilidad con una amplia variedad de dispositivos y hardware, permitiendo al sistema operativo comunicarse correctamente con impresoras, tarjetas gráficas, discos, etc.
